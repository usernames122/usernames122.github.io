Mode

AlignPosition

PositionAlignmentMode

The Mode specifies the way its constraint determines its goal.

OneAttachment

The constraint attempts to move Constraint/Attachment0|Attachment0 to AlignPosition/Position|Position. Constraint/Attachment1|Attachment1 is not used.

TwoAttachment

The constraint attempts to move Constraint/Attachment0|Attachment0 to the position of Constraint/Attachment1|Attachment1. AlignPosition/Position|Position is not used.
